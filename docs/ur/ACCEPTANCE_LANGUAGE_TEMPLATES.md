# Acceptance Language Templates (AER Submissions)

These templates are factual, audit-safe, and do not imply medical necessity or payer endorsement.

## Generic (Default)
- “This submission contains a deterministic Adherence Evidence Report (AER) for the requested period.”
- “Evidence is captured contemporaneously and is reconstructed without inference.”
- “Missing data is explicitly labeled.”

## IOP (Intensive Outpatient Program)
- “This submission supports IOP oversight for a defined seven-day period.”
- “AER evidence is deterministic and based on recorded assignments, responses, and review actions.”
- “Missing data is explicitly labeled.”

## PHP (Partial Hospitalization Program)
- “This submission supports PHP oversight for a defined seven-day period.”
- “AER evidence is deterministic and based on recorded assignments, responses, and review actions.”
- “Escalations (if included) are recorded events, not adjudications.”

## MAT (Medication-Assisted Treatment)
- “This submission supports MAT oversight for a defined fourteen-day period.”
- “AER evidence is deterministic and based on recorded assignments, responses, and review actions.”
- “Missing data is explicitly labeled.”

## Always Include
- “AER does not determine diagnosis, medical necessity, or payer eligibility.”
- “AI outputs are excluded; AER contains only system-of-record evidence.”
