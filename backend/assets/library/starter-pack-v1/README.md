# Starter Library Pack v1

Self-authored starter content for Between Sessions. All items are original templates written by the Between Sessions team. No third-party manuals or copyrighted worksheets are included.

## License
- Type: SELF_AUTHORED
- Source: Between Sessions Starter Pack v1
- Public domain sources: none

## Structure
Each item is a JSON file that follows the schema defined in the ingest script. Every item includes:
- `INSTRUCTIONS`
- `CONTENT`
- `CLINICIAN_NOTES`

All items are client-safe, draft-only, and require clinician review before publish.

## Coverage
- 20 CBT worksheets/prompts
- 15 DBT skills items
- 15 SUD relapse prevention items
- 10 sleep/stress items
- 10 psychoeducation handouts
- 10 self-rating measures (not diagnostic)
- 20 mixed prompts/handouts

Total: 100 items
